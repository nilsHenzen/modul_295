## Start App
- open terminal
- go to folder
- npm i
- nodemon src/server.js / node src/server.js
- open at http://localhost:3000
- /swagger to see endpoints

## Tests
Tests are writen in postman.
Start with the login test, else every other will return 401 and fail.
